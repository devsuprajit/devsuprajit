## Featured Engineering Work

### ESP32 Smart Attendance & Proximity Detection System

> **[devsuprajit/esp32-attendance-system](https://github.com/devsuprajit/esp32-attendance-system)**  
> Cloud-connected IoT attendance prototype with proximity-based submission and a live dashboard.

**The problem:** Make attendance submission proximity-aware while giving an instructor a live, browser-based view of incoming records.

**The approach:** An ESP32 operates in access-point and station modes, serves a local attendance form, and sends JSON records to a Node.js/Express API deployed on Render. The dashboard auto-refreshes to show check-ins and check-outs.

`ESP32` `Arduino C++` `Node.js` `Express.js` `REST API` `HTML` `CSS` `JavaScript` `Render`

[View source](https://github.com/devsuprajit/esp32-attendance-system) / [Open live dashboard](https://esp32-attendance-system.onrender.com)

### Arduino UNO Q Temperature Monitoring System

> **[devsuprajit/arduino-iot-server](https://github.com/devsuprajit/arduino-iot-server)**  
> Device-to-cloud temperature and humidity monitoring with a browser-based live dashboard.

**The problem:** Monitor temperature and humidity beyond the device itself through a simple cloud-accessible interface.

**The approach:** DHT11/DHT22 readings move from Arduino firmware through a Python sender to an Express.js REST API on Render, then appear on an auto-refreshing web dashboard.

`Arduino UNO Q` `C++` `Python` `Node.js` `Express.js` `REST API` `JavaScript` `Render`

[View source](https://github.com/devsuprajit/arduino-iot-server)
